# Módulo 3: Supervivencia en la Terminal

Bienvenido al Módulo 3. Hasta ahora, hemos usado el ratón y las ventanas para movernos, pero ha llegado el momento de abandonar la zona de confort. En este módulo abriremos la consola (terminal) y descubriremos que esa "pantalla negra" no es un castigo, sino la herramienta más rápida, potente y eficiente que existe para controlar tu ordenador.

## 📝 Descripción del Módulo
El objetivo de este bloque es que pierdas el miedo a usar el teclado para comunicarte con el sistema. Aprenderás a moverte, crear archivos, usar atajos que te ahorrarán horas de tecleo y, lo más importante, a pedirle ayuda al propio sistema cuando no recuerdes un comando.

* ⏱️ **Duración estimada del módulo:** 5 horas (La clave aquí es repetir los comandos hasta que se vuelvan memoria muscular).

## 📚 Temas del Módulo
1. **01-anatomia-de-la-consola**: Qué es la terminal, qué es la shell (Bash) y cómo leer el *prompt*.
2. **02-atajos-y-autocompletado**: La tecla TAB es tu mejor amiga. Aprende a ahorrar tiempo, usar el historial y cancelar comandos con `Ctrl+C`.
3. **03-navegacion-de-directorios**: Rutas absolutas vs. relativas. Aprende a ubicarte y moverte usando `pwd`, `ls` y `cd`.
4. **04-gestion-de-archivos**: Toma el control de tus datos. Cómo crear, copiar, mover, renombrar y borrar archivos o carpetas (`touch`, `mkdir`, `cp`, `mv`, `rm`).
5. **05-lectura-y-edicion-de-texto**: Cómo ver el contenido de los archivos sin abrirlos (`cat`, `less`) y cómo editarlos rápidamente desde la consola usando `nano`.
6. **06-pidiendo-ayuda**: No memorices todo. Aprende a consultar los manuales del sistema usando `man` y `--help`.

## 🎯 Resultado Esperado
Al finalizar este módulo, serás capaz de abrir una terminal vacía, navegar con fluidez por tus carpetas, editar documentos y salir de cualquier aprieto buscando información en el propio manual del sistema.
