# Módulo 7: Redes y Servicios Básicos (Conectando con el mundo)

Hasta ahora hemos trabajado "hacia adentro" de nuestra máquina. En este módulo, abriremos las puertas de Linux para comunicarnos con otros ordenadores. Aprenderás a configurar tu conexión, a controlar servidores a miles de kilómetros de distancia y a proteger tu sistema de visitas no deseadas.

## 📝 Descripción del Módulo
Este bloque es esencial para entender cómo funciona internet desde las entrañas de Linux. Veremos desde lo más básico (¿tengo internet?) hasta lo más profesional (administrar un servidor remoto mediante SSH o configurar un cortafuegos). También aprenderemos a gestionar los "servicios", esos programas que corren en silencio en segundo plano.

* ⏱️ **Duración estimada del módulo:** 4.5 horas.

## 📚 Temas del Módulo
1. **01-conceptos-y-comandos-de-red**: Entendiendo la IP, la puerta de enlace y los comandos `ip`, `ping` y `ifconfig`.
2. **02-acceso-remoto-con-ssh**: La herramienta más importante de un SysAdmin. Cómo entrar en otra máquina de forma segura.
3. **03-transferencia-de-archivos (SCP y SFTP)**: Moviendo archivos entre ordenadores como un profesional.
4. **04-el-cortafuegos-ufw**: Cerrando las puertas de tu casa. Cómo permitir o denegar el tráfico de red.
5. **05-gestion-de-servicios-systemd**: Aprendiendo a usar `systemctl` para iniciar, parar y automatizar programas al arrancar el sistema.
6. **06-analisis-de-puertos-y-red**: Uso básico de `netstat` o `ss` para ver quién está conectado a nuestro equipo.

## 🎯 Resultado Esperado
Al finalizar este módulo, serás capaz de configurar la red de un servidor Linux, conectarte a él de forma remota desde cualquier lugar del mundo y asegurar que solo los servicios necesarios estén expuestos a internet.
