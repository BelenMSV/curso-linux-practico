# Módulo 4: Gestión de Paquetes y Software

En Windows, cuando quieres un programa, vas a una web, descargas un `.exe` y rezas para que no tenga virus. **En Linux, eso no funciona así.** En este módulo descubrirás el sistema de "tiendas de aplicaciones" centralizadas que Linux inventó mucho antes que Apple o Google, y aprenderás a gestionar el software de tu equipo tanto a golpe de clic como desde la terminal.

## 📝 Descripción del Módulo
Aprenderás qué son los repositorios, cómo el sistema resuelve "dependencias" y dominarás todos los formatos modernos de instalación. Al final, serás capaz de instalar programas, arreglar instalaciones rotas y mantener tu sistema limpio de archivos basura.

* ⏱️ **Duración estimada del módulo:** 4.5 horas.

## 📚 Temas del Módulo
1. **01-tiendas-graficas-y-repositorios**: Instalando a golpe de clic y entendiendo qué ocurre por detrás (Los almacenes de software y las dependencias).
2. **02-el-comando-apt**: La herramienta definitiva en consola. Aprende a buscar (`search`), actualizar (`update` / `upgrade`), instalar (`install`) y eliminar (`remove`).
3. **03-archivos-deb-y-dpkg**: Cómo instalar los paquetes que descargas manualmente de internet (el equivalente al clásico `.exe`).
4. **04-formatos-universales**: Adiós a los problemas de compatibilidad. Entendiendo cómo usar **Snap, Flatpak y AppImage** (aplicaciones portátiles).
5. **05-ppas-y-repos-externos**: Cómo añadir "almacenes no oficiales" para conseguir las versiones más recientes de tus programas favoritos.
6. **06-mantenimiento-y-reparacion**: Cómo limpiar la basura acumulada (`autoremove`, `clean`) y cómo salvar el sistema si una instalación se rompe a la mitad.
7. **07-compilacion-basica-targz**: [Nivel Avanzado] Qué hacer cuando un programa no tiene instalador y solo te dan el código fuente (`make install`).

## 🎯 Resultado Esperado
Al terminar este bloque, no habrá programa, librería o herramienta que se te resista. Sabrás instalar software desde cualquier fuente, mantenerlo actualizado de forma segura y limpiar los residuos que dejen al desinstalarse.
