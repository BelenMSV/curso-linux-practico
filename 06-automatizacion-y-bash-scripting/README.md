# Módulo 6: Automatización y Bash Scripting (Tu primer robot)

Hasta ahora has aprendido a usar Linux comando a comando. Pero, ¿qué pasa si tienes que crear 100 usuarios, o hacer una copia de seguridad de 20 carpetas todos los lunes a las 8:00 AM? No vas a escribir los comandos uno por uno. Para eso existe el **Bash Scripting**: el arte de guardar secuencias de comandos en un archivo para que el sistema los ejecute automáticamente.

## 📝 Descripción del Módulo
En este bloque daremos el salto de "usuarios" a "desarrolladores de sistemas". Aprenderás la sintaxis básica de los guiones de Bash, cómo usar variables para guardar datos, cómo tomar decisiones lógicas (condicionales) y cómo repetir tareas miles de veces en un segundo (bucles).

* ⏱️ **Duración estimada del módulo:** 5 horas.

## 📚 Temas del Módulo
1. **01-mi-primer-script**: El archivo `.sh`, el "Shebang" (`#!`) y cómo dar permisos de ejecución.
2. **02-variables-y-entrada-de-datos**: Guardando información y haciendo que el script nos pregunte cosas (`read`).
3. **03-operaciones-aritmeticas**: Cómo hacer que Linux sume, reste y calcule por nosotros.
4. **04-condicionales-if-else**: Enseñando al script a tomar decisiones (ej: "Si la carpeta no existe, créala").
5. **05-bucles-for-y-while**: Repetición masiva de tareas sin esfuerzo.
6. **06-funciones-y-organizacion**: Cómo escribir código limpio y reutilizable.
7. **07-proyecto-final-backup**: Creación de un script profesional que comprime una carpeta y le pone la fecha actual automáticamente.

## 🎯 Resultado Esperado
Al terminar este módulo, habrás dejado de ser un operario manual. Serás capaz de identificar tareas repetitivas en tu día a día y escribir scripts que las resuelvan en segundos, ahorrándote horas de trabajo y evitando errores humanos.
