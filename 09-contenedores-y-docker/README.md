# Módulo 9: Contenedores y Docker (El estándar de la industria)

Bienvenido a la revolución de la infraestructura. Si alguna vez has dicho u oído la frase *"Pues en mi ordenador funcionaba"*, este módulo es la solución definitiva a ese problema.

Aprenderemos a usar **Docker**, la herramienta que permite empaquetar aplicaciones completas (con sus bases de datos, servidores web y código) en "contenedores" aislados que funcionan exactamente igual en tu portátil, en un servidor de Amazon o en la Raspberry Pi de tu casa.

## 📝 Descripción del Módulo
En este bloque pasaremos de la administración tradicional a la gestión moderna de infraestructura. Entenderás la diferencia entre una Máquina Virtual pesada y un Contenedor ligero. Aprenderás a descargar imágenes, levantar contenedores, gestionar sus datos para que no se borren y usar Docker Compose para desplegar infraestructuras completas (como el WordPress del módulo anterior) en cuestión de segundos.

* ⏱️ **Duración estimada del módulo:** 5 horas.

## 📚 Temas del Módulo
1. **01-que-es-docker**: Máquinas Virtuales vs Contenedores. Conceptos de Imagen, Contenedor y Registro.
2. **02-instalacion-de-docker**: Preparando nuestro servidor Linux para soportar contenedores.
3. **03-tu-primer-contenedor**: Comandos básicos (`run`, `ps`, `stop`, `rm`) y el clásico "Hello World".
4. **04-persistencia-de-datos-y-redes**: Uso de Volúmenes para que no pierdas tus datos si el contenedor se destruye.
5. **05-creando-tus-propias-imagenes-dockerfile**: Cómo escribir las "recetas" para crear tus propios contenedores personalizados.
6. **06-magia-con-docker-compose**: Desplegando aplicaciones multi-contenedor (WordPress + Base de Datos) con un solo comando.

## 🎯 Resultado Esperado
Al terminar este módulo, serás capaz de destruir y reconstruir servicios complejos en segundos. Ya no tendrás miedo de "romper" tu servidor principal al instalar software, porque todo vivirá encapsulado y seguro dentro de contenedores de usar y tirar.
