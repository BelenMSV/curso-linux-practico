# Módulo 1: Introducción y Entorno

Bienvenido al primer módulo del curso. Antes de escribir comandos o instalar programas, es fundamental entender qué es lo que estamos usando, de dónde viene y cómo preparar un entorno de pruebas seguro para aprender sin miedo a romper nada.

En este módulo sentaremos las bases teóricas y técnicas necesarias para el resto del curso.

## 📝 Descripción del Módulo
Este módulo está diseñado para introducir al alumno en la filosofía del software libre, ayudarle a navegar por la diversidad visual y técnica de Linux y dejar listo un entorno de pruebas virtualizado.

## 📚 Temas del Módulo
1. **01-que-es-linux**: Historia, el concepto de Kernel y la importancia del Open Source en la actualidad.
2. **02-distribuciones**: Qué son las "distros", las familias principales (Debian, Arch, Red Hat) y cómo elegir una.
3. **03-entornos-de-escritorio**: Separando el sistema de la interfaz; exploración de GNOME, KDE Plasma, XFCE y otros.
4. **04-alternativas-de-uso**: Diferencias entre usar un Live USB, Dual Boot, WSL y Máquinas Virtuales.
5. **05-maquinas-virtuales**: Configuración paso a paso de VirtualBox/VMware para crear nuestro laboratorio de prácticas.

## ⏱️ Duración Estimada
Este módulo requiere aproximadamente **3 horas** de dedicación, considerando la lectura de conceptos y la configuración técnica inicial.
