# Módulo 8: Servidores Web y Bases de Datos (El Stack Profesional)

En este punto del curso, tu máquina Linux dejará de ser una herramienta personal para convertirse en un **Servidor de Producción**. Aprenderás a instalar y configurar las piezas de software que permiten que una página web funcione y que los datos se guarden de forma organizada.

Dominaremos el famoso **Stack LEMP**, el estándar más utilizado en la industria moderna por su velocidad y eficiencia.

## 📝 Descripción del Módulo
Aprenderás a instalar un servidor web (Nginx), a gestionar motores de bases de datos (MariaDB/MySQL) y a conectar todo con lenguajes de programación como PHP. Además, veremos cómo hacer que una web sea segura mediante certificados SSL, para que aparezca el famoso candado verde en el navegador.

* ⏱️ **Duración estimada del módulo:** 6 horas.

## 📚 Temas del Módulo
1. **01-introduccion-servidores-web**: ¿Qué es un servidor web? Diferencias entre Apache y Nginx.
2. **02-instalacion-y-configuracion-nginx**: Tu primera web "Hola Mundo" servida al mundo real.
3. **03-bases-de-datos-mariadb**: Instalación, seguridad inicial y comandos SQL básicos.
4. **04-el-stack-lemp**: Uniendo todas las piezas: Linux + Nginx + MariaDB + PHP.
5. **05-seguridad-y-certificados-ssl**: Instalación de Let's Encrypt y HTTPS gratuito.
6. **06-proyecto-final-despliegue**: Instalación de un sistema de gestión de contenidos (CMS) real desde cero.

## 🎯 Resultado Esperado
Al finalizar este módulo, habrás transformado una instalación limpia de Linux en un servidor web profesional, capaz de alojar sitios dinámicos, seguros y preparados para recibir tráfico real de internet.
