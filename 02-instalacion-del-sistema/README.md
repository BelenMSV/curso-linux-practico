# Módulo 2: Instalación del Sistema

En este módulo pasaremos de la teoría a la acción pura. Aprenderás a preparar el instalador, a entender cómo Linux organiza el disco duro y a completar una instalación limpia y funcional. Aunque usaremos una máquina virtual, los pasos que aprenderás aquí son 99% idénticos a los que harías en un ordenador real.

## 📝 Descripción del Módulo
El objetivo de este bloque es que pierdas el miedo al proceso de instalación. Veremos desde la descarga segura de la imagen del sistema hasta la configuración de los primeros parámetros (usuario, zona horaria y particiones). Al finalizar este módulo, tendrás un sistema Linux operativo y listo para recibir comandos.

* ⏱️ **Duración estimada del módulo:** 4 horas (incluyendo tiempos de descarga y espera de la instalación).

## 📚 Temas del Módulo
1. **01-descarga-y-verificacion**: Cómo obtener la ISO oficial de Ubuntu/Mint y verificar que el archivo no esté corrupto (Checksum).
2. **02-preparacion-del-instalador**: Creación de un USB booteable para equipos físicos y configuración del "CD Virtual" en VirtualBox.
3. **03-el-particionado-de-disco**: Conceptos clave: ¿Qué es la partición Raíz (`/`), la `Home` y la `Swap`? Particionado automático vs. manual.
4. **04-instalacion-paso-a-paso**: Recorrido visual por el asistente de instalación, configuración de teclado, red y cuenta de usuario.
5. **05-post-instalacion-basica**: Primer arranque, actualización de repositorios e instalación de "Guest Additions" para una mejor experiencia en máquina virtual.

## 🎯 Resultado Esperado
Tener una instalación de Linux (preferiblemente Ubuntu o Linux Mint) funcionando perfectamente, con acceso a internet y lista para ser administrada.
