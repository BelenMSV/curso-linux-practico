# Módulo 12: Proyecto Final (De Local a la Nube) 🌍

Bienvenido a la prueba definitiva. Todo lo que has aprendido hasta ahora sobre terminal, redes, contenedores, monitorización y seguridad culmina en este preciso momento. 

Un administrador de sistemas o ingeniero DevOps no vive de máquinas virtuales locales; vive de servidores en producción. En este módulo final, vamos a alquilar un servidor real en internet, le asignaremos un nombre de dominio profesional y desplegaremos una infraestructura completa, segura y monitorizada de cara al público.

## 📝 Descripción del Módulo
Pasaremos de "localhost" a la web mundial. Aprenderás qué es un VPS (Virtual Private Server), cómo alquilar uno por unos pocos euros al mes, cómo comprar un dominio (ej. `tu-nombre.com`) y cómo conectar ese dominio a tu servidor mediante registros DNS. Después, aplicarás **todo** el conocimiento de los módulos anteriores para levantar tu proyecto en producción.

* ⏱️ **Duración estimada del módulo:** 5 horas (el tiempo de despliegue dependerá un poco de la propagación de los DNS).

## 📚 Temas del Módulo
1. **01-alquiler-y-acceso-al-vps**: El salto a la nube. Comparativa de proveedores Cloud (DigitalOcean, Hetzner, AWS, Linode), creación de la instancia y primer acceso seguro.
2. **02-dominios-y-registros-dns**: Tu identidad en internet. Cómo registrar un dominio barato y configurar los registros `A` y `CNAME` para que apunten a la dirección IP pública de tu nuevo servidor.
3. **03-checklist-de-produccion**: El ritual del SysAdmin. Una guía paso a paso aplicando los conocimientos de los Módulos 1 al 11 en tu VPS fresco (Actualización, Creación de usuario, Llaves SSH, UFW, Fail2Ban).
4. **04-despliegue-del-proyecto**: Subiendo el código. Migraremos tu Stack LEMP o tu infraestructura de Docker Compose (Nginx + Aplicación + Base de datos) a la nube.
5. **05-https-y-certificados-ssl-reales**: El candado verde. Ahora que tenemos un dominio real, usaremos Let's Encrypt y Certbot para asegurar las comunicaciones de tu web de forma oficial y gratuita.

## 🎯 Resultado Esperado
Al finalizar este módulo, tendrás una página web o aplicación (tu propio portfolio, un blog, o un panel de Grafana) corriendo en un servidor real, accesible desde cualquier parte del mundo con un nombre de dominio profesional, asegurada con HTTPS, fortificada contra ataques y monitorizada en tiempo real. 

Esta infraestructura será la prueba irrefutable de que ya no eres un principiante: **es el proyecto que pondrás en tu currículum o perfil de LinkedIn.**
